Метод файлового объекта close() автоматически закрывает файл, при этом
теряется любая несохраненная информация. Работать с файлом (читать,
записывать) после этого нельзя.
Python автоматически закрывает файл если файловый объект к которому он
привязан присваивается другому файлу. Однако, хорошей практикой будет
вручную закрывать файл командой close().
